# Repository Structure Renaming Update

## Overview

This document records the follow-up repository normalization work completed after the Bun-to-pnpm migration.

## Changes Implemented

### 1. Governance File Rename

- Renamed `CLAUDE.md` to `Agent.md` at the repository root.
- Preserved content continuity while aligning file naming with the updated agent instruction convention.

### 2. Documentation Directory Rename

- Renamed root directory `ai-docs/` to `docs/`.
- Preserved the full documentation tree and content under the new root path.

### 3. Reference Path Updates

- Updated internal file references that previously pointed to `ai-docs/...` so they now point to `docs/...`.
- Updated migration notes in `Docs/feature_2026-04-26_1.0.0.md` to reflect the new path.

## Validation

The following validations were completed:

1. Verified there are no remaining references to `CLAUDE.md`.
2. Verified there are no remaining references to `ai-docs/` paths.
3. Verified `Agent.md` and `docs/` exist at the expected locations.

## Notes

- Existing `ai-docs.md` files inside packages/apps were intentionally not renamed because they are separate per-directory guidance files, not part of the renamed root `ai-docs/` directory.
